# Скрипт для сбора лидов недвижимости - ИНСТРУКЦИЯ

## 📋 Что делает скрипт?

Собирает контактные данные компаний (застройщики, агентства) из Яндекс.Карт по следующим запросам:
- "Недвижимость Москва"
- "Новостройки Москва"
- "ЖК Москва"
- "Застройщики Москва"
- "Агентства недвижимости Москва"

**Собираемые данные:**
- 📛 Название компании
- 📞 Телефон
- 📍 Адрес (если доступен)
- 🌐 Веб-сайт (если доступен)
- 🔍 Поисковый запрос (источник)

## ⚙️ УСТАНОВКА И ЗАПУСК

### Шаг 1: Проверка Python
Убедитесь, что установлен Python 3.8+:

```bash
python --version
```

Если Python не установлен, скачайте с https://www.python.org/downloads/

### Шаг 2: Установка зависимостей

Откройте терминал/командную строку в папке проекта и выполните:

```bash
pip install selenium webdriver-manager
```

Эти пакеты позволяют:
- **selenium** — автоматизация браузера
- **webdriver-manager** — автоматическое скачивание нужного драйвера Chrome

### Шаг 3: Запуск скрипта

**Базовый запуск (по умолчанию 30 результатов на запрос):**
```bash
python scrape_real_estate_leads.py
```

**С параметрами:**
```bash
python scrape_real_estate_leads.py --max-results 50 --headless
```

### Параметры командной строки

| Параметр | Значение | Описание |
|----------|----------|---------|
| `--max-results` | число | Сколько компаний собирать по каждому запросу (по умолчанию: 30) |
| `--headless` | флаг | Запустить браузер без UI (режим "головы") — быстрее, но не видно процесс |

### Примеры команд

```bash
# Стандартный запуск с визуализацией
python scrape_real_estate_leads.py

# Быстрый режим без окна браузера
python scrape_real_estate_leads.py --headless

# Большой объем данных (100 на запрос)
python scrape_real_estate_leads.py --max-results 100

# Комбинированный запуск
python scrape_real_estate_leads.py --max-results 75 --headless
```

## 📊 Ожидаемый результат

После запуска вы увидите:

1. **В консоли (терминале):**
   - Статус скрейпинга в реальном времени
   - Количество найденных компаний
   - Сообщение об успешном сохранении

2. **В файле `leads.csv`:**
   - CSV-таблица с собранными данными
   - Заголовки: Компания, Телефон, Адрес, Сайт, Поисковый запрос

## ⏱️ Время выполнения

- **Без headless** (с визуализацией): ~3-5 минут
- **С --headless** (быстрый режим): ~2-3 минуты

Время зависит от:
- Скорости интернета
- Количества результатов (--max-results)
- Загруженности серверов Яндекс.Карт

## ⚠️ ВАЖНЫЕ ЗАМЕЧАНИЯ

### Правовые аспекты
1. ⚖️ **GDPR / ФЗ о персональных данных (РФ)**
   - Собираемые данные — это публичная информация о компаниях (не личные данные людей)
   - Для коммерческого использования проверьте локальные законы

2. 📜 **Условия обслуживания Яндекс.Карт**
   - Используйте скрипт в рамках Terms of Service
   - Не перегружайте серверы частыми запросами

3. 🔄 **robots.txt**
   - Скрипт соблюдает разумные задержки между запросами (2 сек.)
   - Не используйте экстремально частые запросы

### Технические аспекты
1. 🌐 **Интернет-соединение** — требуется стабильное подключение
2. 🖥️ **Chrome должен быть установлен** — скрипт использует Chrome браузер
3. 🔄 **Изменения на сайте** — если Яндекс.Карты обновит дизайн, селекторы может потребоваться обновить

## 🔧 Решение проблем

### Ошибка: "No module named 'selenium'"
```bash
pip install selenium webdriver-manager
```

### Ошибка: "Chrome driver not found"
- Убедитесь, что Chrome установлен: https://www.google.com/chrome/
- Скрипт автоматически скачивает драйвер (требуется интернет)

### Ошибка: "TimeoutException"
- Возможна медленная скорость интернета
- Увеличьте время ожидания (измените значение в коде при необходимости)

### Нет результатов или пустой CSV
- Возможны временные проблемы с Яндекс.Картами
- Попробуйте запустить позже
- Проверьте интернет-соединение

### Браузер открывается и закрывается очень быстро
- Это нормально в режиме `--headless`
- Используйте без `--headless` для визуализации процесса

## 📈 Повышение производительности

Если нужно собрать много данных:

1. **Используйте --headless:**
   ```bash
   python scrape_real_estate_leads.py --headless --max-results 100
   ```

2. **Запускайте несколько раз с разными запросами:**
   - Данные дедупликируются автоматически

3. **Расширьте SEARCH_QUERIES в коде:**
   - Отредактируйте список поисковых запросов в скрипте
   - Добавьте новые города или категории
